Automatic Grade Generator Using Fuzzy Logic



WE are here to help ease the process of grading. evaluating papers is itself a hard task, so is generarting grades from those marks This platform adds a human factor to the grading process, taking into account things like attendence, importance of an evaluation component and its complexity with the help of Fuzzy Logic

You only need to follow theese four steps:

1.) name the xlsx file that you have marks in as "marks.xlsx"
2.) the sheet should have the following columns: (ID, Name, Attendence, Quiz, Lab, Mids, Compre)
3.) in the following page, please enter the maximum marks for each component chosen.
4.) if the course you have doesnot have any of the above components, put maximum marks as zero, and all the values under that column in "marks.xlsx"
5.) sit back and get the grades
